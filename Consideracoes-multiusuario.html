Considerações multiusuário

O acesso multiusuário é facilitado com um modelo cliente/servidor. Obviamente, queremos isso. Também precisamos de leitura e escrita simultâneas do mesmo banco de dados. Isso é feito bloqueando partes ou todo o banco de dados enquanto eles são atualizados. É preciso decidir sobre a unidade de dados a ser bloqueada. EU ( TBL) imagine que seria um nó, não um banco de dados.
